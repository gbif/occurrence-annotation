# Get a project

Get a project

## Usage

``` r
get_project(id = NULL, offset = NULL, limit = NULL)
```

## Arguments

- id:

  the id of the project.

- offset:

  page start.

- limit:

  number of records to return on page.

## Value

a \`tibble\`.

## Examples

``` r
if (FALSE) { # \dontrun{
get_project()
} # }
```
