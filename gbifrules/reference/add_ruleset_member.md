# Add a new member to a ruleset

Add a new member to a ruleset

## Usage

``` r
add_ruleset_member(id = NULL, new_member = NULL)
```

## Arguments

- id:

  the ruleset id.

- new_member:

  the GBIF user to be added.

## Value

A \`list\` of ruleset information.

## Examples

``` r
if (FALSE) { # \dontrun{
add_ruleset_member(1,"JOHN")
} # }
```
