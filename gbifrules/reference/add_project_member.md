# Add a new member to a project

Add a new member to a project

## Usage

``` r
add_project_member(id = NULL, new_member = NULL)
```

## Arguments

- id:

  the project id.

- new_member:

  the GBIF user to be added.

## Value

A \`list\` of project information.

## Examples

``` r
if (FALSE) { # \dontrun{
add_project_member(1,"JOHN")
} # }
```
