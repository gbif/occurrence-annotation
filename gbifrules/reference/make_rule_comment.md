# Make a comment on rule.

Make a comment on rule.

## Usage

``` r
make_rule_comment(id = NULL, comment = NULL)
```

## Arguments

- id:

  id of the rule you want to leave a comment on.

- comment:

  comment.

## Value

A \`tibble\`.

## Examples

``` r
if (FALSE) { # \dontrun{
make_rule_comment(1,"comment")
} # }
```
