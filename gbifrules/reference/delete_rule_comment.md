# Delete a rule comment

Delete a rule comment

## Usage

``` r
delete_rule_comment(ruleId, commentId)
```

## Arguments

- ruleId:

  The ID of the rule that the comment belongs to

- commentId:

  The ID of the comment to delete

## Value

Nothing (invisible NULL)

## Examples

``` r
if (FALSE) { # \dontrun{
delete_rule_comment(ruleId = 1, commentId = 5)
} # }
```
