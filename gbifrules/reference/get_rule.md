# Get rules

Get rules

## Usage

``` r
get_rule(id = NULL, limit = NULL, offset = NULL, ...)
```

## Arguments

- id:

  rule id.

- limit:

  page start.

- offset:

  number of records to return on page.

- ...:

  Additional args passed to query.

## Value

a tibble of rules

## Examples

``` r
if (FALSE) { # \dontrun{
get_rule()
} # }
```
