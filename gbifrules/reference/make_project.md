# Make Project

Make Project

## Usage

``` r
make_project(name = NULL, description = NULL)
```

## Arguments

- name:

  the name of the project.

- description:

  describe the project.

## Value

The projectId of the project.

## Examples

``` r
if (FALSE) { # \dontrun{
make_project("Example project","An example Project")
} # }
```
