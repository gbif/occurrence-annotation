# Get a ruleset.

Get a ruleset.

## Usage

``` r
get_ruleset(id = NULL, projectId = NULL, limit = NULL, offset = NULL)
```

## Arguments

- id:

  the id of the ruleset.

- projectId:

  id of project.

- limit:

  page start.

- offset:

  number of records to return on page.

## Value

a \`tibble\`.

## Examples

``` r
if (FALSE) { # \dontrun{
get_ruleset(1,1)
} # }
```
